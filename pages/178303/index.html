这里主要封装了Reponse工具类，用来处理一些响应的问题。

# Response::getAddress

获取当前访问的URL域名

# Response::getRootDomain

获取当前访问的根域名

# Response::getRootDomain

获取域名

# Response::getNowAddress

获取当前访问的地址

# Response::getMyIp

获取服务器IP

# Response::msg

跳转提示

参数 类型 解释
$err boolean 是否发生错误
$code int 响应代码
$title string 错误标题
$msg string 错误信息
$time int 跳转超时，设置为-1不跳转
$url string 跳转地址
$desc string 跳转描述

注意

采用API方式构建网站的也可以使用该方法。

笔记

跳转提示的 tpl 模板位于 src/static/innerView/tip/common.tpl ，可以自行修改。

# Response::location

直接跳转不提示

参数 类型 解释
$url string 跳转地址
$timeout int 跳转超时，设置为0直接跳转

# Response::isInner

是否为内网IP

参数 类型 解释
$ip string 需要判断的IP

# Response::getHttpSchema

获取当前使用的协议，例如http://

# Response::setHeader

设置响应头

参数 类型 解释
$key string 键名
$name string 键值
